# Journey-Hub
Journey Hub is a responsive travel website that helps users explore destinations, discover hotels and restaurants, and plan trips with ease. Featuring search, filters, and attractive listings, it provides a simple and engaging travel planning experience across all devices.
# Journey Hub - Travel Website
## Features

* Responsive design for desktop, tablet, and mobile devices
* Navigation bar with menu links
* Search section for destinations
* Featured destinations section
* Popular hotels and restaurants listings
* Card-based layout with images and descriptions
* CSS hover effects and modern styling

## Technologies Used

* HTML5
* CSS3
